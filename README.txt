Module permettant d'ajouter le transporteur Bring4You à votre installation Woocommerce

== Description ==

Le module Bring4You permet de livrer vos colis encombrants à moindre coût

== Installation ==

1. Envoyer un email à hello@bring4you.com pour obtenir la clé d’API
2. Avoir installé Woocommerce sinon l'installation échouera.
3. Envoyer le module vers le répertoire `/wp-content/plugins/` ou directement avec WordPress
4. Activer le module dans le menu 'Plugins' de WordPress
5. Configurer le module
6. Placer le shortcode '[b4y_estimation]' dans votre page produit.

N'hésitez pas à  suivre notre guide d'installation :
https://stonly.com/guide/fr/integration-wordpress-b0oIbHd4Xl/Steps/

Notre vidéo de démonstration :
https://www.youtube.com/watch?v=zMofmyOcwq0
